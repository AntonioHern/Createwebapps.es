Proyectos

Admin Ajax

Proyecto desarrollado en Laravel, con PHP y Javascript.

Es un proyecto que consiste en una aplicación web para la gestión de usuarios de una residencia, para llevar el control de sus medicinas y la preparación de sus diferentes pastilleros.

Consta con un sistema de avisos por notificaciones vía email para avisar cuando a algún usuario le queda poca medicina.

También tiene un sistema de mensajería interna entre usuarios de la aplicación.

Dialog Bot

Dialog Bot es un proyecto desarrollado en Node utilizando la herramienta de Google, DialogFlow, que permite crear un chatbot para una página web.

Puede almacenar datos en una BBDD nosql como MongoDB, consta de un lenguaje fluido y capaz de entender el contexto del cual se le está hablando

Covid

Aplicación desarrollada en Laravel con PHP y Javascript, consiste en una serie de gráficas por países más relevantes para ver datos de casos de Covid 19 tanto de infectados, muertes y recuperaciones

La aplicación obtiene los datos de una API externa y se actualiza diariamente sin necesidad de ninguna clase de mantenimiento
